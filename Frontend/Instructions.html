Wiki Installer v2.0 - Justin

Introduction

Welcome to the new version of the Wiki Installer! The old one was kind of a mess, so I just revamped it a bit, especially in the backend. The Wiki Installer is, as the name suggests, a tool to easily download all the items from the Public Forum wiki over the past years. It's meant to be like a one-stop shop sort of thing when it comes to obtaining open-sourced cases, blockfiles, etc.

To run the app, go to the default directory (The one that houses the package.json) and type 'node .'

Filtering

Enter the date numbers in MM/DD/YYYY format! This will prompt the backend software to perform a search and display all matching results on the table. You may also enter matching query phrases in the "name" column that will match the files against the entered names. Keep in mind they are space sensitive.

Downloading

You could download files directly from the browser page. Just enable permissions. Upon completion, you'll see that it changes into a check mark.

Desktop Instant Download

Alternatively, we can do things the old fashioned way by instantly downloading onto the desktop via the node app. This will, however, restrict your download to the device that's hosting the backend software. Click the desktop instant download and the files will appear on C://Downloads/...
